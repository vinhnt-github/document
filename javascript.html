JavaScript

Hoisting

Hoisting is JavaScript's default behavior of moving all declarations to the top of the current scope (to the top of the current script or the current function).

Do not apply "let" and const 

x = 5; // Assign 5 to x

elem = document.getElementById("demo"); // Find an element
elem.innerHTML = x;                     // Display x in the element

var x;
hoisting convert to
var x; // Declare x
x = 5; // Assign 5 to x

elem = document.getElementById("demo"); // Find an element
elem.innerHTML = x;

var,let,const

var

var have scope is function

var x = 10;
// Here x is 10
var x = 2;
// Here x is 2
Function scope
var y = 10;
// Here y is 10
function myFunc {
  var y = 2;
  // Here y is 2
}
// Here y is 10
let,const

The let,conts keyword allows you to declare a variable with block scope (block scope is {} but not {} of object )

let(or const) x = 10;
{
    let(or const) x = 2;
    // Here x is 2
}
// Here x is 10

Bind()

The bind()là phương thức tạo ra 1 hàm mới,khi được gọi, từ khóa this của nó được đặt thành giá trị được cung cấp cho chuỗi các đối số bất kỳ được cung cấp nào khi hàm mới được gọi.

Syntax : function.bind(this)

const module = {
    x: 42,
    getX: function() {
        return this.x;
}
};
const myObj = {
    x: 100,
};

const unboundGetX = module.getX; //unboundGetX = function() {return this.x;}
console.log(unboundGetX()); // The function gets invoked at the global scope
// expected output: undefined

const boundGetX = unboundGetX.bind(module);
console.log(boundGetX());// expected output: 42
const myObjGetX = unboundGetX.bind(myObj);
console.log(myObjGetX());// expected output: 100
call()

call() là phương thức gọi 1 hàm với giá trị của this và các đói số riêng lẻ của nó

Syntax : function.call(this,arguments1,arguments1...)

const module = {
    x: 42,
    getX: function() {
        return this.x;
}
};
const myObj = {
    x: 100,
};

const unboundGetX = module.getX; //unboundGetX = function() {return this.x;}
console.log(unboundGetX()); // The function gets invoked at the global scope
// expected output: undefined

const boundGetX = unboundGetX.bind(module);
console.log(boundGetX());// expected output: 42
const myObjGetX = unboundGetX.bind(myObj);
console.log(myObjGetX());// expected output: 100
apply()

apply() là phương thức gọi 1 hàm với giá trị của this và các đói số của nó là 1 mảng

Syntax : func.apply(thisArg, [ argsArray])

const numbers = [5, 6, 2, 3, 7];

const max = Math.max.apply(null, numbers);

console.log(max);
// expected output: 7

const min = Math.min.apply(null, numbers);

console.log(min);
// expected output: 2
